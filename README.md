
# Simple Calculator

This is a basic Python calculator that supports addition, subtraction, multiplication, and division. It provides a simple command-line interface for users to perform calculations.

## Usage

1. Run the `calculator.py` script.
2. Enter the first and second numbers when prompted.
3. Choose the operation: Addition, Subtraction, Multiplication, or Division.
4. The result will be displayed.

## Example

```bash
python calculator.py
